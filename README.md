# 🛒 쇼핑 리스트 앱

한국어 쇼핑 리스트 웹 앱입니다. 별도의 서버나 설치 없이 브라우저에서 바로 사용할 수 있습니다.

## 기능

- 아이템 추가 (엔터키 또는 추가 버튼)
- 체크박스로 완료 표시
- 개별 아이템 삭제
- 완료 항목 일괄 삭제
- localStorage로 데이터 영구 저장
- 부드러운 애니메이션 효과

## 사용법

`shopping-list.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
